
Напиши программу на питону, которая будет решать квадратное уравнение: ax**2 + bx + c = 0. Такую что:
1) В ней НЕ будет использована конструкция TRY EXCEPT
2) Решение уравнение будет корректным
3) Программа не должна падать
4) При D > 0, 2 два корня, при D = 0 - два одинаковых корня, при D < 0 корней нет.
5) В твоем блоке кода не должно быть внутри принтов. Чтобы можно было юзать как метод из библиотеки ты ему туда кидаешь значения, он возвращает результат.
6) Все ошибки были обработаны и если вдруг что-то идет не так пользователь бы получал понятное сообщение ошибке

Входные данные задачи: коэффициенты а, b, c  - вещественные числа
Выходные данные: корни уравнения: x1, x2.

Формулы:
D = b**2 - 4 * a * c
x1 = (-b + sqrt(D)) / (2 * a)
x2 = (-b - sqrt(D)) / (2 * a)
